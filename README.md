# ML-resume-screening-system
Project Overview:
The AI Resume Screening System is designed to automate candidate evaluation by matching resumes against job descriptions. It uses Natural Language Processing (NLP), skill extraction, TF-IDF similarity scoring, and ATS-inspired ranking techniques to identify the most suitable candidates for a role.

*Key Features:
1.Resume parsing from PDF and TXT files
2.Automated skill extraction using NLP
3.Job description analysis
4.TF-IDF and Cosine Similarity matching
5.Weighted skill-based scoring
6.ATS-style candidate ranking
7.Critical and recommended skill gap analysis
8.Automated hiring recommendations
9.Export screening results to CSV

*Technologies Used:
1.Python
2.Pandas
3.NumPy
4.Scikit-learn
5.spaCy
6.NLTK
7.PDFPlumber
8.TF-IDF Vectorization
9.Cosine Similarity

*Machine Learning Workflow:
1.Resume Text Extraction
2.Text Cleaning & NLP Preprocessing
3.Skill Extraction
4.Job Description Analysis
5.TF-IDF Similarity Calculation
6.Weighted Skill Matching
7.Composite Candidate Scoring
8.ATS Decision Generation
9.Candidate Ranking & Reporting

*ATS Decision Categories:
⭐ Strong Match
✅ Good Match
🔶 Average Match
⚠️ Below Average
❌ Not Suitable

*Business Applications:
1.Recruitment Automation
2.Talent Acquisition
3.HR Analytics
4.Applicant Tracking Systems (ATS)
5.Resume Shortlisting
6.Candidate Screening

*Model Benefits:
1.Reduces manual resume review time
2.Improves hiring efficiency
3.Provides objective candidate evaluation
4.Identifies missing critical skills
5.Enhances recruitment decision-making
6.Scales to large applicant pools

Results:
The system ranks candidates based on resume-job relevance, highlights critical skill gaps, and generates ATS-style recommendations to help recruiters identify the most qualified applicants quickly and accurately.
